PHP - AJAX form

                
Baikal has a built-in universal php-ajax form that can grab dynamic amounts of inputs and send it to any given email address. To make the form work, simply add the class .zform like <form class="zform">.

                
You can also specify where to send the data via an hidden input:
<input type="hidden" name="to" value="username@domain.extension">

                
Other reserved name fields are: subject , email , name
